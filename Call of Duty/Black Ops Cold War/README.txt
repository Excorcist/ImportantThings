  2. Unlock all tool (Unlock all is the SUA.dll)
Note: The unlock all tool is only soft unlock, which means everything will go back to normal after the game restarts.

Instructions for injecting Soft Unlock Tool:
  Download SUA.dll
  Download process hacker and run through the installation
  Run Call of Duty Black Ops Cold War (Steam or Bnet, Works for both)
  Run process hacker
  Find Black Ops Cold War process and right click it
  Find miscellaneous, open that then find Inject dll and select that
  Once the file viewer tab pops up find SUA.dll and select it
  After that boom, it's injected. Enjoy your soft unlock
